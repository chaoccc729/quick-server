# quick-server
Takeway App
运行项目

**先运行服务器，再打开客户端**

**canyeshop-server文件夹下cmd命令：**

1. `npm install`
2. `npm start`

**canyeshop-client文件夹下cmd命令：**

1. `npm install`
2. `npm start`

**登陆**

1. 手机号登陆，输入符合格式的手机号即可
2. 密码登陆，默认用户名abc，密码123
